# FeatureRenderer

The program “FeatureRenderer” symbolizes all existing entries in the 
UniqueValueRenderer (individual value classification) of a given ArcMap Layer. 
Therefore the UniqueValueRenderer should contain (if desired?) all available values 
related to the legend table - with „AddAllValues“ added automatically. 
Legend entries can be excluded by manual preselection in the UniqueValueRenderer of the Layer. 

## Install

Binary downloads of the ArcMap plugin can be found at the following links:
* [Windows](http://gisgba.geologie.ac.at/LegendGenerator/) - FeatureRenderer.esriAddIn

Execute the file "FeatureRenderer.esriAddIn".
